Important Analyst Notes

  • This is a first-pass screen, not a full investment decision.
  • Also review free cash flow growth, debt-to-equity, margins, insider ownership, competitive moat, industry cyclicality, macroeconomic risk, and management quality.
  • Some sectors naturally fail certain tests. Banks, utilities, REITs, ETFs, and mutual funds may require different evaluation standards.
  • High P/E stocks are not automatically bad if earnings growth, moat, and margins justify the premium.
  • Low P/E stocks are not automatically good if revenue, profitability, or balance sheet quality is deteriorating.

Backend / Proxy Explanation

This website sends ticker requests to your Render backend. Render then securely contacts the Financial Modeling Prep API using your private API key. This prevents your API key from being exposed in browser JavaScript.
